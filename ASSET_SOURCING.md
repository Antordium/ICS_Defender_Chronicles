## Asset Modification Guide

### Recoloring Sprites for Characters

Using GIMP or similar:

1. Open sprite sheet
2. Colors → Hue-Saturation
3. Adjust hue to match character color scheme:
   - CIPHER: Shift toward blue (180°)
   - BLAZE: Shift toward red/orange (0-30°)
   - GHOST: Shift toward purple (270°), reduce saturation
   - VOLT: Shift toward yellow (60°)
4. Export as PNG

### Creating Glitch Effects

For GHOST and hacker enemies:
1. Duplicate sprite layer
2. Offset RGB channels slightly
3. Add noise
4. Create animation frames with varying offsets

### Adding Neon Glow

For cyberpunk aesthetic:
1. Select bright areas
2. Expand selection 2-3 pixels
3. Feather
4. Fill with bright color at low opacity
5. Set layer to "Screen" or "Add"

## Custom Asset Creation Notes

If creating custom pixel art:

**Character Sprites:**
- Size: 16x24 or 24x32 pixels
- Colors: 16 color palette per character
- Animations: Idle (2f), Walk (4f), Attack (3f)

**Tilesets:**
- Tile size: 16x16 pixels
- Format: 256x256 sheet (16x16 tiles)
- Include: Floor, walls, objects, transitions

**Portraits:**
- Size: 64x64 (small) or 128x128 (detailed)
- Expression variants: Neutral, Happy, Angry, Surprised
